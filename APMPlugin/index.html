First make sure your project is saved as the name you want to see in the document

Second, make sure your sequence starts at time 0 and the desired end point is the end of the last clip in your timeline.

Third, select the type of production type your video will be.



Fourth, click generate which will then prompt you to pick the file you want the info saved to. Make sure you choose a .txt file. If you don't have one, open a text editor and create one
